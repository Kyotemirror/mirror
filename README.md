<p align="center">
  <img src="mirror/assets/logo.png" width="200">
</p>

A lightweight, Raspberry Pi–friendly **Smart Mirror dashboard** built with **Python and Pygame**.  
Designed for reliability, simplicity, and subtle branding.

---

## ✨ Features

- 🕒 **Centered Clock**
- 📅 **Current Day & Date**
- 🌦 **Live Weather** (OpenWeather API)
- 🎨 **Custom Branding Logo** rendered into the background
- ⚡ Fast startup, low overhead
- 🧱 No web browser, no Electron, no heavy frameworks
